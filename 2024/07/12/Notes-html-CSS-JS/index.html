变量与常量

  • 变量声明

    变量声明不用强调其类型，可以let number = 25, str = "25"; 甚至可以转身就给之前赋数值的变量赋一个字符串

  • 常量声明

​ 常量声明用const代替let即可，其值一旦声明不可改变，声明时必定要赋值

  • 变量类型

    • Number类

      包括整形和浮点数，存在Infinity和NaN两个特殊数据

      • Infinity可由1/0得到，也可以直接调用该变量
      • NaN表示运算出错，一旦出现，则其相关所有运算结果都是NaN
      • 运算中，求幂的符号是**，允许小数次幂；单个+用在变量前有类型转换（为Number）的作用；存在+=系列、自增自减、位运算全家桶，但多出一个>>>无符号右移。

    • BigInt类

      ​ 表示绝对值在253−12^{53}-1253−1范围外的大数，类似于long long，声明时在值后面加一个’n’（例如const bi = 2147483647n）

    • String类

      • 允许用单引号（‘hello’），双引号（“hello”），反引号（`hello`）进行包含
      • 其中前两种近似等价，最后一种允许使用${}将表达式或变量包装嵌入字符串，表达式会被计算，例如：

      ​ 会弹出一个显示3的弹窗。

      • 类比cpp中的string，可以使用加号连接起来，且如果加法运算一个操作数是字符串，则结果就是字符串。

    • Boolean类（同cpp)

    • null值

      不属于任何类型，不像cpp那样作“空指针”之用

    • undefined值

      声明但未被赋值则为undefined

    • Object类

    • Symbol类

  • 可以使用typeof运算符（不是函数）查看参数的类型，使用方法同cpp。

  • 在算数函数/表达式中，自动将变量转换为number

  • 可以使用String(var)的方式进行类型转换

  • 转换为布尔时，若值为空则为false，否则true

    ​ 这就说明，如果将字符串“0”进行转换，得到值是true。

  • 不同类型比较时会进行类型转换（转为number)，这导致其无法直接区分0和false，需要使用严格相等===与严格不相等!==进行比较

    • 特殊情况：null == undefined

交互函数

  • alert：弹窗一个，只有确定按钮，该弹窗弹出期间不能与其他东西交互（被称为模态窗，下面两种函数的弹窗也是模态窗）

  • prompt：接收title和default两个参数，前者为显示的文本，后者为默认输入的值（可以没有），弹出一个有输入框的窗口，如果点取消返回null，否则为输入的内容（字符串）

  • confirm：接收一个参数Question，返回值为布尔

  • switch…case…

  • for…in
